# FlowMesa

## O que é o FlowMesa

O FlowMesa é um sistema de operação para WhatsApp, balcão, PDV, caixa, estoque e impressão por setor.  
Ele foi pensado para centralizar pedidos, organizar a produção e facilitar o atendimento em negócios de alimentação.

## Problema que ele resolve

Muitos negócios atendem por WhatsApp e balcão ao mesmo tempo, mas usam processos soltos, sem integração entre pedido, produção, caixa e estoque.  
O FlowMesa resolve isso com um fluxo único e organizado.

## Para quem é

O sistema é indicado para:

- food trucks
- lanchonetes
- balcões de atendimento
- operações com retirada
- negócios que vendem por WhatsApp e presencialmente

## Módulos principais

- WhatsApp
- PDV de balcão
- pedidos
- senha de retirada
- impressão por setor
- caixa
- estoque
- relatórios
- configurações

## Fluxo resumido do pedido

1. O cliente faz o pedido pelo WhatsApp ou no balcão.
2. O sistema registra o cliente e os itens.
3. O pedido é separado por setor.
4. Os tickets são enviados para impressão.
5. O pagamento é registrado.
6. O estoque é baixado automaticamente.
7. O pedido é finalizado e acompanhado até a entrega.

## Planos

### Base
Indicado para operações menores, com foco em WhatsApp, pedido e impressão básica.

### Pro
Indicado para quem precisa de PDV, caixa, estoque e impressão por setor.

### Premium
Indicado para operações mais completas, com múltiplas unidades, relatórios avançados e personalização.

## Objetivo do projeto

Criar uma plataforma SaaS completa para organizar vendas, atendimento, produção e operação em um único sistema.
